Funções importantes:

Babel: converter (transpilar) código do React para um código que browser entenda

Webpack: Para cada tipo de arquivo (.js, .css, .png) eu vou converter o codigo 
de um maneira diferente


Loaders: babel-loader, css-loader, image-loader é feita para conseguir entender arquivos
dando sentido ao codigo JS


JSX: HTML dentro do JS, traduzindo (Javascript XML)
